Todo list - без UI

Примеры взаимодействия

  • Функция changeStatus("write a post", "Done") - будет менять статус задачи.
    Доступные статусы: 'In Progress', 'Done', 'To Do'.
  • Функция addTask('have a walk') - добавляет новую задачу
  • Функция deleteTask('have a walk') - удаляет задачу
  • Функция showList() будет выводить весь список дел.
